listy {wbgcharts}R Documentation

Create a list with self-referential elements

Description

listy() is like list() except that successive elements can refer back to previously defined elements

Usage

listy(...)

Arguments

...

objects, possibly named.

Examples

# This doesn't work
list(a = 2, b = 2 * a)

# But this does
listy(a = 2, b = 2 * a)
