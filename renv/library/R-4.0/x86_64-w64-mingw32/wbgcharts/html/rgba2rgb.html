rgba2rgb {wbgcharts}R Documentation

Convert an RGBA color to RGB by combining with a background color

Description

Convert an RGBA color to RGB by combining with a background color

Usage

rgba2rgb(color, background = "white", alpha = NULL, preserve.names = TRUE)

Arguments

color

a color or vector of colors

background

a background color

alpha

the alpha to use, which if not NULL will override any alphas in color

preserve.names

preserve the names of color if any
